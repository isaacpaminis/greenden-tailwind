Greenden – Responsive Website (HTML & Tailwind CSS)

Overview:

Greenden is a responsive multi-page website built using HTML5 and Tailwind CSS.
The project focuses on creating a clean, modern, and user-friendly interface using Tailwind’s utility-first styling approach.
It demonstrates real-world layout structure and responsive design techniques.

Features:

Responsive navigation bar

Multi-page website structure

Grid-based product layout

Clean and modern UI

Fully responsive across all screen sizes

Pages Included: 

Home – Hero section with introduction

Products – Plant-themed product cards in a responsive grid

Contact – Structured contact information layout

Responsive Design:

The website adapts seamlessly to mobile, tablet, and desktop screens using Tailwind responsive classes (sm:, md:, lg:).

Technologies Used:

HTML5

Tailwind CSS
